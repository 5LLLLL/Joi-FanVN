# 序章-Joi

在日常下播后，Joi突然昏迷，来到了宇宙女皇的宫殿；面对发生异常的各个星球，Joi接受了宇宙女皇的请求。
# 第一章-Military

再次醒来，Joi受到了身着绿色军装的轴伊和轴伊小队的照顾。在月色下听过轴伊的哼唱，Joi提议在这片大地寻找八音盒，播放纸带里留下的旋律。
于是白天，小队向着可能留有零件的城市废墟前进。
如同奇迹一般，小队找到了生锈但能用的八音盒，简单修复后，八音盒里传来了断断续续的旋律。
突然，一颗流弹打破了宁静。紧急之中，轴伊决定牺牲自己掩护队员撤离；Joi抗拒了她的命令，劝说她一起撤离：如果没了她，小队也会分崩离析的。
在夜色之下，小队成功撤离，轴伊握着手中的托帕石，望着天上的群星。
# 间章
# 第二章-Flowers

Joi这次降临到了一片森林，没走几步便遇见了一位老轴芯。老轴芯领着Joi来到了村子，暂住在老轴芯家。Joi尝试打听与自己长相相似的人，无果。
夜晚，Joi有些睡不着，出去散步（或是听到了熟悉的曲调），遇到了一只小狐狸。Joi想追上狐狸，却意外迷失在了森林中。正当她不知所措时，有着对狐狸耳朵的轴伊出现在她面前，带Joi来到了一座破旧的古庙。
在了解到轴伊和古庙的事后，Joi决定培养轴伊成村里的小偶像，重新获得轴芯们的愿力，复兴古庙。
在经历一系列事情后，古庙前的花海再次得到绽放。

# 第三章-Snow

来到这里，Joi感到异常寒冷，这是一片雪中的小村落。在路边转悠时，Joi被教会误认为是雪轴伊而被带走。在教会里遇到雪轴伊，解开误会后，教会想请Joi离开，雪轴伊却把Joi留了下来。通过询问得知雪轴伊身为圣女，正在以宣扬教义为由周游这个世界，只为找到关于“海”的线索。
轴伊身边的骑士轴芯跟她一样来自共同的家乡——轴芯村。Joi从轴芯口中了解到，轴伊很久没回家乡了，轴芯们都很想她。为了到秘密基地里了解这个世界与自己直播企划中的差别，也为了轴芯们的思念，Joi劝说轴伊回趟家乡。
回到轴芯村，轴伊自然地收到了轴芯们的嘘寒问暖。夜里轴伊终于抽出空闲，带着Joi来到了秘密基地。这里被保养的很好，连西贝的弦都刚被更换。在轴伊的弹唱中，Joi意外被告知任务完成了。次日白天，便与众人告别离开。
# 第四章-Library

荒芜，是Joi来到这里的第一反应。不知为何，Joi没有回到星空，周围依然下着大雪。雪中矗立着一座巨大的建筑。Joi走进其中，发现周围摆放着密密麻麻的图书。图书馆内十分黯淡，中央蜷缩着一位轴伊。
